# Apple Stock Trend & Algorithmic Trading Signal

Objective: Developed a Python-based trading algorithm to detect momentum crossovers using 30, 50, and 120-day Moving Averages.

Key Features:

Data: Sourced via yfinance API.

Logic: Implemented a "Confluence Strategy" requiring multiple trend alignments before triggering a signal.

Analysis: Calculated strategy accuracy and total return vs. Buy & Hold benchmark.

Tools Used: Python, Pandas, Matplotlib, NumPy.
